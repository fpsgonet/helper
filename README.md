# 欢迎来到 fpsgo.net 辅助工具仓库

## 📌 介绍

本仓库提供适用于 [fpsgo.net](https://fpsgo.net) 的油猴 (Tampermonkey) 脚本，帮助用户优化游戏体验。

## 📂 文件说明

请将以下脚本安装到油猴 (Tampermonkey) 以启用外挂功能：

- **修改昵称-1.0.user.js**：允许用户自定义昵称，**注意：不能与隐身同时开启**。
- **加速-1.0.user.js**：提供人物移动和射击加速功能。
- **隐身-1.0.user.js**：其他人在对局内看不到你的人物昵称和模型。

## 🚀 安装方法

1. **安装油猴 (Tampermonkey)**
   - 访问 [Tampermonkey 官网](https://www.tampermonkey.net/) 并安装适用于您的浏览器的扩展。
2. **安装脚本**
   - 打开本仓库，找到 `修改昵称-1.0.user.js`、`隐身-1.0.user.js`、`加速-1.0.user.js`。
   - 点击每个文件，选择 **Raw** 按钮。
   - 在打开的新页面中，Tampermonkey 会弹出安装提示，点击 **安装**。
3. **启用辅助**
   - 确保 Tampermonkey 和上述脚本处于启用状态。
   - 进入 [fpsgo.net](https://fpsgo.net) 享受辅助功能！
